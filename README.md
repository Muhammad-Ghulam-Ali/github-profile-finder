# GitHub Profile Finder

A clean, real-time GitHub profile search app built with React + Tailwind CSS.

## Features

- Search any GitHub username instantly
- View profile info — name, bio, followers, following, public repos
- Browse the 5 most recently updated repositories
- Direct links to GitHub profile and individual repos
- Loading state with animated indicator
- Error handling for invalid usernames

## Tech Stack

- React
- Tailwind CSS
- GitHub REST API

## Getting Started

```bash
npm install
npm run dev
```

## API Used

[GitHub REST API](https://docs.github.com/en/rest) — no auth token required for public data.
